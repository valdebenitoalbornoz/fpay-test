# Steps to run

1. Clone the project:

```git clone https://github.com/valdebenitoalbornoz/fpay-test.git```

```npm install```

```npm run test```

```docker pull danielvaldebenito/fpay-test```


5. Run docker image

```docker run fpay-test -p 3000:3000 -it```
