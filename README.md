# Steps to run

1. Clone the project:

```git clone https://github.com/valdebenitoalbornoz/fpay-test.git```

2. Install dependencies:

```npm install```

3. Run locally

```npm start```

4. Build docker image

```docker build . -t fpay-test```


5. Run docker image

```docker run fpay-test -p 3000:3000 -it```


6. Testing

```npm run test```
